# Acquire Font Setup

**To use the Acquire font:**

1. Download the font from: https://www.fontspace.com/aquire-font-f43735
2. Place the font files in this directory (`src/app/fonts/`):
   - `Aquire-Regular.otf` (required)
   - `Aquire-Bold.otf` (optional, for bold text)
3. Open `src/app/globals.css` and uncomment lines 16-34 (the @font-face blocks)
4. Build and run the site

**Currently:** The site uses system fallback fonts until you add the Acquire font files.
